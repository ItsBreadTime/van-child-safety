# PCB

PCB schematic, layout, and manufacturing exports.

| File | Purpose |
|---|---|
| `schematic.pdf` | Schematic export |
| `pcb.pdf` | PCB layout render |
| `pcb_top.pdf` | Top-side render |
| `pcb_bottom.pdf` | Bottom-side render |
| `gerber.zip` | Gerber manufacturing package |
| `pcb_easyeda.json` | EasyEDA-native editable source |
| `pcb_3d.obj` | Generated 3D board export |

Review all footprints, pin assignments, regulator/current limits, mounting holes, and connector clearances before manufacturing.
