# Hardware Design

Hardware source and manufacturing/export files for the Van Child Safety prototype.

| Subdir | Source tool | Contents |
|---|---|---|
| `pcb/` | EasyEDA | Schematic, PCB render, Gerbers, EasyEDA JSON |
| `enclosure/` | Onshape | STEP/STL enclosure exports and assembly render |

## Notes

The PCB and enclosure files are provided as prototype artifacts. Validate footprints, clearances, power, thermal behavior, sensor placement, and enclosure materials before adapting them for a real vehicle.
